# 前言
本系列是《计算机系统要素》的读书笔记，也是 Nand2Tetris 课程的配置教材。

Nand2Tetris 即从与非门到俄罗斯方块，本书将告诉你，如何从与非门开始一步一步构建计算机，如何实现一门语言和相应的语言标准库，实现一个简单的编译器，如何实现一个简单的操作系统，以及如何在这样的操作系统上开发应用软件，比如俄罗斯方块。
