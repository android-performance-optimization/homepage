## 元编程的基本概念
### 元编程的定义
元编程(Meta Programming)，就是以程序为数据的编程。这意味着元编程往往是以访问和修改程序本身为目的的。其中，编写元程序的语言被称为元语言，被操作的语言则被称为目标语言。
